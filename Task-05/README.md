A responsive **Hero Section landing page** for a Laundry Service website built using **HTML and CSS**.

This project is part of the **Task 5: CSS Hero Section (Display, Position, Float)** from the **Tutedude MERN Stack Web Development course**.

The goal of this task is to practice **CSS layout techniques** and create a clean hero section similar to modern service landing pages.

---

# 🚀 Features

✔ Responsive Navigation Bar
✔ Clean Hero Section Layout
✔ Call-to-Action Button
✔ Laundry Service Illustration
✔ Mobile Friendly Design
✔ Uses Viewport Units
